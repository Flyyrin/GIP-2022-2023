Het Project

Welkom bij ons dambord project. Ons doel is om het klassieke damspel naar het digitale tijdperk te brengen met dit geavanceerde ontwerp. Het bord wordt bestuurd door twee joysticks, waardoor spelers een interactieve spelervaring krijgen. Bovendien hebben spelers toegang tot, en controle over het bord via een gebruiksvriendelijke webinterface, waardoor het voor iedereen gemakkelijk is om het spel te overzien. De LED-lampjes, die zowel de stukken als het bord verlichten, voegen een vleugje hightech verfijning toe aan het spel en spelers kunnen hun ervaring aanpassen met verschillende kleuropties. Met zijn innovatieve ontwerp en geavanceerde technologie belooft ons LED Checkers Board bestuurd door joysticks en een webinterface een game-changer te worden in de wereld van checkers. Maak je klaar om je damspel naar een hoger niveau te tillen!

Programmeertalen

We gebruiken Python op de Raspberry Pi om het damspel te laten werken, LED-dambord aan te sturen, de joystick en knoppen te laten werken, de webinterface weer te geven en onze server te vertellen wat die moet doen bij het krijgen en geven van data.

Voor de website hebben we JavaScript gebruikt om verschillende onderdelen te programmeren. Zo hebben we bijvoorbeeld een responsieve image map toegevoegd, die zich aanpast aan de grootte van het scherm waarop de website wordt bekeken. Ook gebruiken we dit om de gespeelde games te laten zien op onze website.

Om de website te maken, hebben we gebruik gemaakt van HTML. Dit is een belangrijke taal voor webontwikkeling, waarmee de structuur van de website wordt gedefinieerd en de inhoud wordt weergegeven. Hierdoor is onze website overzichtelijk en gemakkelijk te navigeren.

Naast HTML hebben we ook CSS gebruikt om onze website te stylen naar onze wensen. CSS is een belangrijke taal voor het definiÃ«ren van de visuele stijl en lay-out van een website. Dit geeft onze website een professionele uitstraling en maakt het aantrekkelijk voor bezoekers.

We gebruiken Json om instellingen en configuraties gemakkelijk bij te houden en om de speldata op onze server op te slaan. Json is een handige taal voor het opslaan en delen van gegevens tussen verschillende systemen en applicaties. Hierdoor kunnen we ons project gemakkelijk aanpassen en configureren, en hebben we altijd toegang tot onze speldata.

Python Modules

We gebruiken de json module om onze gegevens te coderen en decoderen. zodat we deze kunnen lezen, aanpassen, verzenden naar en opslaan op onze server.

neopixel is Een module die wordt gebruikt voor het aansturen van de NeoPixel-LED-strip, die individueel adresseerbare RGB-leds bevat. Hiermee kunnen we elke tegel van ons dambord apart aansturen.

board is en module die wordt gebruikt om de pinconfiguratie van de Raspberry Pi in te stellen. Het helpt ons om externe apparaten aan te sluiten met de gpio pinnen.

We gebruiken de queue module om gegevens te delen tussen threads of processen. Hierdoor kunnen wij gegevens delen tussen de webinterface en het damspel.

De threading module die wordt gebruikt om meerdere threads in een Python-programma te maken. Hiermee kunnen we het damspel en de webinterface parallel uitvoeren.

requests is een module die wordt gebruikt om HTTP-verzoeken te sturen en te ontvangen. We gebruiken het om spelgegevens te versturen naar onze server.

time is een module die wordt gebruikt om tijd-gerelateerde functies uit te voeren. Hiermee kunnen we vaststellen om welke tijd een spel wordt gespeeld.

os is een module die toegang biedt tot enkele functies van het besturingssysteem. We gebruiken deze module om de locatie van configuratie bestanden te bepalen.

Met de subprocess module kan je externe commando's uitvoeren. Hiermee kunnen we onze webinterface sluiten.

De webview module is een module die een eenvoudige manier biedt om webgebaseerde gebruikersinterfaces te maken in Python. Hierdoor kunnen we een webinterface maken die eruitzien en werken als een normale website, maar data kan doorgeven aan ons python progamma.
